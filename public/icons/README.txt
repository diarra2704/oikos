Icônes PWA pour Oikos
=====================
Ajoutez ici deux fichiers PNG pour que l’app soit installable sur mobile :
- icon-192x192.png  (192×192 px)
- icon-512x512.png  (512×512 px)

Vous pouvez générer ces tailles à partir du logo (ex. public/images/logo.png) avec un outil en ligne (realfavicongenerator.net, etc.) ou avec un logiciel de retouche.
